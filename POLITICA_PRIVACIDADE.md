# 🔒 Política de Privacidade - Hooda

**Última atualização:** 28 de Junho de 2026

---

## 1. Introdução

A Hooda ("nós", "a gente", "a plataforma") respeita a tua privacidade. Esta política explica como recolhemos, usamos, guardamos e protegemos os teus dados pessoais.

**Se não concordas, não uses a plataforma.**

---

## 2. Que Dados Recolhemos?

### 2.1 Dados que TÚ nos dás:
- ✅ Nome completo
- ✅ Email
- ✅ Foto de perfil
- ✅ Bio / Descrição
- ✅ Localização (opcional)
- ✅ Redes sociais (opcional)
- ✅ Conteúdo que publicas (posts, vídeos, histórias, livros)

### 2.2 Dados que recolhemos automaticamente:
- 📍 Localização (se permitires)
- 🕐 Hora dos acessos
- 📱 Tipo de dispositivo
- 🌐 IP address
- 🔍 Pesquisas que fazes
- ❤️ Conteúdo que gostas / guardas
- 💬 Mensagens (encriptadas)

### 2.3 Dados de terceiros:
- Se te logas com Google / Apple, recebemos o teu email
- Se partilhas conteúdo com amigos, recolhemos esse histórico

## 4. Quem Pode Ver os Teus Dados?

### 🔓 Dados públicos (todos veem):
- Nome
- Username
- Foto de perfil
- Bio
- Posts / Vídeos / Histórias
- Seguidores / Seguindo

### 🔒 Dados privados (só tu vês):
- Email
- Número de telefone (se adicionares)
- Histórico de downloads
- Livros guardados
- Mensagens privadas

### 🤝 Partilhamos com:
- **Supabase** (base de dados) — seguro, GDPR compliant
- **Google Analytics** (estatísticas) — sem dados pessoais
- **Stripe** (pagamentos) — se usares (em futuro)
- **Não partilhamos** com redes sociais, publicidade, ou terceiros sem consentimento

## 5. Quanto Tempo Guardamos os Teus Dados?

| Tipo de Dado | Tempo | Opção de Apagar? |
|---|---|---|
| **Perfil** | Enquanto tiveres conta | ✅ Apagar conta = apagar tudo |
| **Posts** | Até apagares | ✅ Podes apagar a qualquer hora |
| **Mensagens** | Até apagares | ✅ Ambos podem apagar |
| **Histórias** | 24 horas (auto-delete) | — |
| **Livros guardados** | Enquanto guardares | ✅ Remove quando quiseres |
| **Backups** | 30 dias | ✅ Pedido manual |
| **Logs de acesso** | 90 dias | ❌ Para segurança |

## 6. Segurança

### 🛡️ Como protegemos:
- ✅ Encriptação end-to-end nas mensagens
- ✅ Passwords com hash (nunca armazenadas em texto)
- ✅ HTTPS em tudo
- ✅ 2FA (autenticação de 2 fatores)
- ✅ Backups automáticos
- ✅ Firewalls e DDoS protection

### ⚠️ O que NÃO fazemos:
- ❌ Não vendemos dados
- ❌ Não rastreamos em outras websites
- ❌ Não pedimos dados que não precisamos
- ❌ Não damos acesso a APIs de terceiros

---

## 7. Os Teus Direitos (GDPR)

Tens o direito de:

1. **Acesso** — pedir cópia de todos os teus dados
2. **Correção** — editar dados incorretos
3. **Apagar** — "direito ao esquecimento" — apagar tudo
4. **Portar** — receber teus dados num formato aberto
5. **Recusar** — não concordar com certos usos
6. **Reclamar** — contactar a autoridade de proteção de dados

**Como exercer:** Email para hooda.aoa@gmail.com
